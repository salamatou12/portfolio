### 
Objectifs du projet

Maîtriser la structure HTML et l’organisation d’une page complète.

Approfondir la mise en page moderne avec Flexbox et Grid.

Intégrer des animations simples et interactives.

Créer un site responsive adapté à tous les écrans.

Respecter les bonnes pratiques en développement front-end.

## Technologies utilisées

HTML5

CSS3 (Flexbox, Grid, animations)

Google Maps Embed

Responsive Design

## Principales balises HTML utilisées + Significations

header : zone supérieure contenant le menu/navigation.

section : structure chaque partie de la page (about, services…).

nav : barre de navigation.

main : contenu principal du site.

article: contenu autonome (cartes de services ou projet).

footer: pied de page.

form : formulaire de contact.

input / textarea: champs du formulaire.

img: affichage des images du portfolio et de la section About.

button : actions de formulaires ou interactions.

## Expérience personnelle 

Ce projet m’a permis d’améliorer ma compréhension de HTML/CSS de manière concrète. J’ai appris à structurer une page complète de manière propre et organisée. L’intégration de Flexbox et Grid m’a aidé à obtenir un design moderne et responsive. Les animations m’ont permis d’ajouter une touche d’interactivité. Enfin, ce portfolio représente une base solide que je peux améliorer au fil du temps.
